New env: `AYB_AUTH_OAUTH_RETURN_TO_ALLOWLIST` (host-only allowlist entries; no wildcard path matching). `validatedOAuthReturnTo` in `internal/auth/handler_oauth.go` is the single security owner and enforces HTTPS targets except explicit `localhost` / `127.0.0.1` HTTP allowances, with validation at both OAuth start (`handleOAuthRedirect`) and callback dispatch (`dispatchOAuthCallbackResponse`).
